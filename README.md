# SixStar Sports Website

Sleek football-focused NIL agency website built with Next.js.

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize Roster

Edit `app/page.tsx` and update the `rosterSpots` array with each player:
- name
- position
- bio

Player cards are already styled and ready for image + bio updates.

## Logo

The uploaded logo is stored at `public/logo.png`.

## Push to GitHub

```bash
git init
git add .
git commit -m "Initial SixStar Sports website"
git branch -M main
git remote add origin https://github.com/<your-username>/<your-repo>.git
git push -u origin main
```
